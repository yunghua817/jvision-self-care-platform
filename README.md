# Jvision 美業預約營運平台 Demo

可互動測試的沙龍、SPA、美療、醫美、按摩、指甲店與多據點品牌營運平台。

## 功能

- 自助預約與智慧排班
- 客戶檔案、偏好與諮詢表單
- POS 結帳、訂金、會員點數
- 會員方案與回訪訊息
- 多店營運儀表板
